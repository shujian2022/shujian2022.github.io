HTML常用标签

HTML概述
发布日期:   2022-03-15

一、前端UI技术（静态网页技术）

HTML : 超文本标记语言，用于对网页内容进行结构化的语言

CSS: 层叠样式表， 用于对网页进行美化作用的语言

JavaScript: 脚本语言，用于给网页添加动态行为的语言

jQuery: JavaScript封装库，并提供更加强大、更简洁的API

后期的更高阶的技术

​ Vue.js 前端渐进式的框架

​ node.js 后台服务器

​ webpack : 前端工程化

​ Vue-cli : 脚手架

​ 前端组件库：Element-UI / LayUI(jQuery+Bootstrap)

二、HTML

2.1特点：

​ 1. 标记语言

​ 2. 结构化的语言

2.2语法：

​ 双标签： <标签名称></标签名称>

​ 单标签 : <标签名称>

2.3HTML网页有一个最基本的结构

​ 浏览器在解析一个HTML页面时，首先会判断这个网页的HTML是什么版本，然后使用相应的解析器去解析HTML网页中的各个标签，当解析到不同标签时就浏览器会进行相应的处理。

​ <html></html> 标签表示网页接收的开始和结束

​ <head></head> 可以在这对标签内设置网页的元信息

​ 比如：给搜索引擎提供的关键字，描述，进行搜索引擎优化的内容； 可以设置网页的标题； 可以设置网页的MIME类型和编码

​

2.4 主流开发HTML的IDE

  1. Sublime
  2. Webstorm (IDEA一家公司)
  3. HbuilderX(数字天堂)
  4. vscode(微软)

2.5 HTML常用标签

HTML标签按照在文档流中的特点划分

2.5.1块级元素：

      1. 独占一行：块级元素的宽度默认和父元素一致
      2. 块级元素默认可以通过css样式width和height属性设置块元素的宽度和高度
标签名称 标签作用 特点
p 段落标签，文本分段 上下有空白行
div 盒子标签，用于布局 可以div与div嵌套
header 作用上等价div,但是这个标签具有语义
强调这个标签包含的内容是网页内容的头部，比如顶部导航		 HTML5出来的标签
footer 作用上等价div,但是这个标签具有语义
强调这个标签包含的内容是网页内容的脚部，比如底部版权声明		 HTML5出来的标签
nav 作用上等价div,但是这个标签具有语义
强调这个标签包含的内容是网页中的导航部分		 HTML5出来的标签
section 作用上等价div,但是这个标签具有语义
强调这个标签包含的内容是网页中的独立区域		 HTML5出来的标签
aside 作用上等价div,但是这个标签具有语义
强调这个标签包含的内容是网页中的侧边栏		 HTML5出来的标签
ul 无序列表 有默认的样式
ol 有序列表 有默认的样式
dl 自定义列表标签
li li标签是列表项标签，配置ul，ol使用 有默认的样式
dt 自定义列表项标题标签,配置dl使用
dd 自定义列表项标签,配置dl使用
table 表格标签：通常用于表格布局
th 表格标题标签
tr 表格行标签
td 表格的单元格标签
form 表单标签
h1~6 内容的标题标签，对包含的文本进行加粗加大，1到6组件减小 有默认的样式

2.5.2 行内（行级）元素

  1. 行级元素没有独占一行
  2. 行级元素的宽度和高度默认与内容的宽高一致，无法通过CSS属性width和height设置
标签名称 标签作用 特点
span 通常通过这个标签对文本进行样式的添加 很纯粹的标签
a 用于进行网页跳转的 有默认的样式和行为
font 可以对文本字体样式进行一些设置 不建议使用，希望通过CSS设置字体样式
input 表单元素标签

2.5.3 行内块元素

  1. 没有独占一行
  2. 默认可以通过CSS样式width和height设置宽度和高度，而且行内块元素之间有默认间距，而且没法设置这个间距
标签名称 标签作用 特点
img 关联图像的标签

2.3.4 标签的属性

所有的标签都有 id属性、name属性、style属性、class属性

​ cellspacing属性： 设置单元格和边框、单元格和单元格之间的间隙
